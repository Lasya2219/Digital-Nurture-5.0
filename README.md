# Cognizant DeepSkilling

This repository contains my solutions, hands-on exercises, and mini-projects completed as part of the **Cognizant Digital Nurture (DeepSkilling) Program**. It covers Java fundamentals, backend development, database programming, testing, Spring Framework, Spring Boot, and Microservices.
